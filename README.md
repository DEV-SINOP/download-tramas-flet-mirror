# 🛰️ Download Tramas

O **Download Tramas** é uma ferramenta moderna e intuitiva desenvolvida para simplificar a coleta, análise e visualização de rastros (tramas) de equipamentos. Construído com foco em performance e experiência do usuário, ele substitui ferramentas legadas por uma interface fluida e recursos avançados de geoprocessamento.

---

## ✨ Principais Funcionalidades

- **🚀 Interface Moderna:** Navegação intuitiva e rápida construída com tecnologia Flet.
- **📥 Download Inteligente:** Busca e baixagem otimizada de arquivos de tramas diretamente dos servidores.
- **📊 Dashboard de Status:** Acompanhe o progresso de tarefas e o status operacional em tempo real.
- **🗺️ Visualizador de Mapas:** Visualize os rastros capturados em mapas interativos, permitindo análise precisa de trajetórias.
- **📄 Relatórios Automáticos:** Gere relatórios consolidados em formatos prontos para uso e compartilhamento.
- **🔍 Filtros Avançados:** Encontre exatamente o que você precisa filtrando por data, cliente, equipamento ou tipo de evento.
- **🔄 Atualizações Automáticas:** O app verifica e instala novas versões sozinho, garantindo que você tenha sempre os recursos mais recentes.

## 🚀 Como Usar

Para começar a usar o Download Tramas, siga os passos abaixo:

1. **Baixe o Executável:**
   Acesse o [Site Oficial / Espelho](https://dev-sinop.github.io/download-tramas-flet-mirror/) e faça o download da versão mais recente para Windows.

2. **Configuração Inicial:**
   Ao abrir o app pela primeira vez, insira suas credenciais e configure os caminhos de salvamento na aba de **Configurações**.

3. **Busque e Baixe:**
   Utilize o painel de filtros para selecionar os dados desejados e clique em baixar. O app cuidará de todo o processo de download e organização.

4. **Analise e Exporte:**
   Use o **Visualizador de Tramas** para checar os rastros no mapa e gere relatórios através do botão de exportação.
